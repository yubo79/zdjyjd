# 中德有线 ZDJYJD 分流脚本

## 脚本简介

`zdjyjd.ps1` 是一个 PowerShell 网络配置脚本，用于在 Windows 系统中实现**双网卡分流**：
- **有线网卡（以太网）**：走内网流量
- **无线网卡（WiFi）**：走外网流量

---

## 功能特性

### 1. 查看网络信息 (`Get-NICInfo`)
显示所有已连接网络适配器的详细信息：
- 网卡名称、描述、MAC地址
- IPv4地址、子网掩码、默认网关
- DNS服务器地址
- 网卡索引（InterfaceIndex）

### 2. 一键设置 (`One-Key-Setup`)
自动完成以下配置：
1. 连接 WiFi（ZDJYJD）
2. 设置网卡优先级（有线条先）
3. 添加静态路由规则
4. 验证DNS解析
5. 测试网络连通性

### 3. 还原初始设置 (`Restore-Default`)
恢复默认网络配置：
1. 删除静态路由
2. 恢复网卡默认跃点
3. 重新配置以太网
4. 刷新DNS缓存
5. 断开WiFi

## 网络架构

### IP配置（以太网）
| 项目 | 值 |
|------|-----|
| IP前缀 | 172.23.*.* |
| 子网掩码 | 255.255.248.0 (/23) |
| 默认网关 | 172.23.7.254 |
| DNS服务器1 | 192.168.11.150 |
| DNS服务器2 | 192.168.11.158 |

### 分流规则
| 流量类型 | 网段 | 网卡 | 网关 |
|---------|------|------|------|
| 内网流量 | 172.23.0.0/16, 192.168.0.0/16 | 以太网 | 172.23.7.254 |
| 外网流量 | 0.0.0.0/0 (默认路由) | WiFi | 172.16.191.254 |

### 优先级设置
- 有线网卡（以太网）Metric: 10
- 无线网卡（WiFi）Metric: 20

数值越小优先级越高，确保有线网络优先。

## 使用方法

### 前置要求
- 以**管理员权限**运行 PowerShell
- 计算机需要有有线网卡和无线网卡

### 运行步骤
1. 以管理员身份打开 PowerShell
2. 进入脚本所在目录
3. 执行脚本：
```powershell
.\zdjyjd.ps1
```

### 菜单选项
| 选项 | 功能 |
|------|------|
| 1 | 一键设置 - 自动配置内外网分流，同时连接有线和WiFi |
| 2 | 还原初始设置 - 恢复默认配置，只使用有线网络 |
| 3 | 退出 |

## 技术细节

### 关键函数

| 函数名 | 作用 |
|--------|------|
| `Get-NICInfo` | 获取并显示所有已连接网卡的详细信息 |
| `Get-MatchingIP` | 查找符合内网前缀(172.23.*.*)的IP地址 |
| `Set-Metric-Silent` | 静默设置网卡跃点（优先级） |
| `Add-Routes-Silent` | 添加/删除静态路由规则 |
| `Connect-WiFiProfile` | 通过XML配置连接WiFi网络 |
| `Test-DNS` | 验证内网和外网DNS解析 |
| `Verify-Config` | 测试内网和外网连通性 |

### WiFi配置
- 网络名称（SSID）：`ZDJYJD`
- 安全类型：WPA2-PSK
- 加密方式：AES
- 密码：`zdjyjd@123`

### DNS验证
- 内网域名：`jac.net`
- 外网域名：`baidu.com`

## 注意事项

1. **管理员权限**：脚本必须以管理员权限运行（`#Requires -RunAsAdministrator`）
2. **双网卡要求**：需要有可用的有线网卡和无线网卡
3. **网线连接**：查看网络信息时需要插入网线
4. **静默操作**：设置函数使用 `-ErrorAction SilentlyContinue` 抑制错误输出
5. **外网出口**：WiFi作为外网出口，需要确保WiFi可以访问互联网

---

## 适用场景

- 企业/学校内网环境，需要同时访问内网资源和外网资源
- 内网使用固定IP段（172.23.*.*），通过有线连接
- 外网通过指定的WiFi网络访问
